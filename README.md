# Black Sea Urea Market Analysis

## Overview
This project explores price volatility and seasonal patterns in the Black Sea urea market using data-driven methods.  
Rather than focusing on price prediction, the analysis examines uncertainty, assumptions, and the limits of modeling in volatile commodity markets.

## Research Questions
- What volatility patterns exist in the Black Sea urea market?
- Are there identifiable high-risk periods based on historical data?
- How do modeling assumptions influence conclusions in volatile markets?

## Data
- Historical urea price data (Black Sea region)
- Time period: To be specified
- Frequency: Weekly or monthly

All data is anonymized and used solely for educational and analytical purposes.

## Methods
- Exploratory data analysis (EDA)
- Volatility metrics (rolling variance, percentage change)
- Baseline statistical models
- Optional machine learning classification (high vs low volatility periods)
